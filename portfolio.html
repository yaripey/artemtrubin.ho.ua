Medzdrava

Medzdrava is a web application, created for a cardiology in Zaporizhzhya. Unfortunately, project was canceled mid-production, but the app is almost finished. The app itself is a patient-medical-card system, where doctors can create events in patient cards, make notes and conclusions.

The project features Flask based site with some static pages and a system of role-based profiles. Admins can also create posts that non-loged users can view on the main page in a slider. Those a written in Markdown.
